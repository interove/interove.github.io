# Hello World 👋

欢迎来到我的个人博客！

这是我用 **React + Vite + Tailwind CSS** 搭建的静态博客，部署在 GitHub Pages 上。

## 关于这个博客

这个博客将会记录：

- 🛠️ **技术文章** — 编程、工具、架构思考
- 📖 **读书笔记** — 好书精华与个人感悟
- 🌱 **成长日记** — 学习过程中的点滴记录
- 💡 **想法碎片** — 随手记录的灵感与思考


```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print([fibonacci(i) for i in range(10)])
```
